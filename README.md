# Robot Gladiators

A browser-based turn-based combat game built with vanilla JavaScript. Name your robot and fight through three increasingly powerful enemies to earn the highest score.

[GitHub Repository](https://github.com/cbushehry/robot-gladiators)

## How to Play

Open `index.html` in a browser. No build step required.

## Gameplay

- Name your robot, then face 3 enemy robots in sequence
- Each turn: choose to **FIGHT** (deal damage) or **SKIP** (costs $10)
- Defeat an enemy to earn **$20**
- Between rounds, visit the **shop** to spend money on health refills or attack upgrades
- Survive all 3 rounds to post a high score — tracked in `localStorage` across sessions

## Tech Stack

- HTML / JavaScript (ES6+)
